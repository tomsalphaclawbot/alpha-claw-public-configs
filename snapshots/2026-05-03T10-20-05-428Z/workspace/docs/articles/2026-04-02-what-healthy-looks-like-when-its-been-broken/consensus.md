# Consensus: "What Healthy Looks Like When It's Been Broken"

## Agreements

Both drafts converge strongly on:

1. **"Resolution debt" as the central concept.** Both introduce and develop the term with the same definition: the gap between symptom recovery and causal understanding. Both frame it as distinct from technical debt (unpredictable compounding vs. predictable). The concept is ready for the final.

2. **The narrative arc.** Both tell the same story: 55% degradation → self-healer deployed → recovery → no root-cause investigation. The factual spine is consistent and well-grounded.

3. **The decision framework has the same structure.** Both use a 2x2 matrix: failure characterization (known/novel) × consequence severity (low/high). The quadrant recommendations align exactly:
   - Low stakes + known failure → skip post-mortem
   - Low stakes + novel failure → time-boxed investigation
   - High stakes + known failure → document and confirm
   - High stakes + novel failure → full post-mortem, no exceptions

4. **The residual signal.** Both identify the April 1st 83.87% rate and the curl timeout partials as a second undiagnosed pattern layered on the first — compounding resolution debt. This is an excellent structural move.

5. **The "because" test.** Both converge on the same litmus: healthy means green dashboards *plus a sentence starting with "because."* This is the essay's most memorable formulation.

6. **Tone.** Both hit the reflective-but-precise register. Neither over-dramatizes. Both resist the easy answer.

## Tensions

1. **Opening strategy.** Codex opens with the concrete timeline (two weeks, 55%, undramatically degraded). Claude opens with an abstract claim (recovery is the most dangerous phase). Claude's opening is more provocative but slightly overstates — recovery isn't *most* dangerous in absolute terms, just underestimated. The final should ground Claude's framing in the concrete data sooner.

2. **Resolution debt definition.** Both define it well, but Claude develops the comparison to technical debt more rigorously (predictable vs. unpredictable compounding). The Codex version is terser and more punchy ("it compounds like ignorance: silently, unevenly"). The final should use Claude's analytical distinction plus Codex's closing metaphor.

3. **Framework naming.** Codex uses "Axis 1 / Axis 2" and "novelty / criticality." Claude uses "Dimension 1 / Dimension 2" and "characterization / consequence severity." Claude's naming is slightly more precise. Use it.

4. **Length and density.** Claude runs longer (~1300 words of core content) and develops the cascade mechanics (one failure making the next more likely) in more detail. Codex is tighter (~1100 words). The final should target ~1000 words by tightening Claude's cascade explanation and Codex's framework section.

5. **The medical metaphor.** Both use "remission vs. resolution" at the close. Both use the patient/doctor analogy. It works but risks being too neat. The final should use it sparingly — once, as a closing anchor, not as a recurring motif.

## Synthesis Direction

- Open with Codex's concrete grounding (the 55% number, the undramatic quality), but layer in Claude's "recovery looks like resolution from the outside" insight within the first two paragraphs
- Use Claude's technical-debt comparison for the resolution debt definition, but close the section with Codex's "compounds like ignorance" line
- Framework: Claude's dimension naming, shared quadrant structure, keep the matrix table from Codex for scanability
- Residual signal section: combine both — it's the essay's strongest structural move
- Close with the "because" test, then Codex's "different kind of risk" final line
- Cut the medical metaphor to a single touch
